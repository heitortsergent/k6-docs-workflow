# cookieJar()

Get the active cookie jar.

| Type                                                                    | Description         |
| ----------------------------------------------------------------------- | ------------------- |
| [CookieJar](https://grafana.com/docs/k6/<K6_VERSION>/javascript-api/k6-http/cookiejar) | A CookieJar object. |

### Example

{{< code >}}

```javascript
import http from 'k6/http';

export default function () {
  const jar = http.cookieJar();
}
```

{{< /code >}}
